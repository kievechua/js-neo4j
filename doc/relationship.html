Upon successful creation of a relationship, the new relationship is returned. Details

createRelationship: createRelationship = (fromNodeId, toNodeId, type, properties) -> params = to: "#{@url}/db/data/node/#{toNodeId}" type: type data: properties utils.post("#{@url}/db/data/node/#{fromNodeId}/relationships", params) cRelationship: createRelationship
